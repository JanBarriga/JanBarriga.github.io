  • Una línia (dash) és igual a tres punts
  • L'espai entre les parts d'una meteixa lletra és igual a un punt (dot). (S = ···, l'espai entre cada punt és un punt)

  • L'espai entre dues parules és igual a set punts (Pa ······· tomàquet)

Sense punters: Javascript i python

La funció dels punters es tenir un control total per part del programador de la memoria del ordinador y recursos del sistema. Peró es molt facil provocar errors de memoria o accesos no permesos si no s'utilitzen bé. Es a dir, llegeix el contigut d'una adreça a la memoria i la posició de l'adreça per manipular-la. Hi ha posicions especials per a una adreça, per exemple en el sectors d'arrencada. Donant la possibilitat de reparar-les. Es fa utilitzar l'ampersand (&) o asterisc (*) per indicar pointers. per tenen funcions diferents Ampersand et permet saber l'adreça del pointer, mentres que el asterisc indica contingut de l'adreça de la memoria. La memoria d'un ordinador es pot imaginar com a caixes d'un array, on cada caixa te una posició concreta. que pot ser buit o amb un valor concret.

Les operacions avançades que pot fer els punters son:

  • Accés directe a la memoria
  • Permet modificar valors d'una posició especifica
  • Es la millor manipulació possible de strings (cadenes) o arrays

    • L'array és, en esencia, una llista d'adreçes limitades amb principi i final.

  • Pasar gran quantitats de dades a funcions

    • En lloc de copiar valors ds passa la seva adreça, estalviant memoria i temps. Aquest es un dels motius pel cual C es mes rapid que python o javascript

  • Creació d'estructures dinamiques facilment

    • Llistes enllaçades a la memoria o grafics.

  • Interacció facil amb microcontroladors

    • Els microcontroladors son sistemes amb llenguatxe de baix nivell perque es mes facil de manipular, encara que ha sortit micropython o circuit python
